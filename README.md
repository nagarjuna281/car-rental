# Car Rental Website UI

A modern and responsive **Car Rental Website User Interface** built using **HTML, CSS, and JavaScript**.  
This project was created by following a YouTube tutorial to practice and strengthen frontend development skills.

---

## 🚀 Features
- Sleek hero section with highlighted vehicle (Porsche Taycan)
- Search functionality for location, start date, and return date
- Trending vehicles section with pricing
- Top rentals showcase with categories (Luxury, Sports, Electric, Hybrid)
- Subscription form with discount offer
- Responsive design for different screen sizes

---

## 🛠️ Tech Stack
- **HTML5** – Structure and semantic markup
- **CSS3** – Styling, layout, and responsiveness
- **JavaScript (ES6)** – Interactivity and dynamic elements

---

## 🎯 Learning Outcomes
- Practiced building a complete UI from scratch
- Improved understanding of layout structuring
- Learned how to integrate design with functionality
- Strengthened frontend development fundamentals
